## Montenegrin language support

SweetSpot now supports **26 languages** with the addition of Montenegrin.

### What's new

- **Montenegrin translation** — a dedicated Montenegrin translation in Latin script with Ijekavian pronunciation, so Montenegro users no longer fall back to Serbian Cyrillic.
- **Language subtitle in Settings** — when using "System default", the Settings screen now shows the actual system language name underneath, matching the language picker.
