{{< changelog version="5.3" date="5. dubna 2026" >}}
- Roční předplatné nahrazuje jednorázový nákup — 14denní bezplatná zkušební doba zůstává beze změny
- Aplikace znovu ověří stav předplatného při návratu do popředí
- Vylepšení překladů v maďarštině, rumunštině, polštině, bulharštině a černohorštině
{{< /changelog >}}

{{< changelog version="5.2" date="2. dubna 2026" >}}

- Výběr ikon nyní zobrazuje název vybrané ikony přeložený do všech 25 podporovaných jazyků
{{< /changelog >}}

{{< changelog version="5.1.5" date="1. dubna 2026" >}}

{{< /changelog >}}

{{< changelog version="5.1.3" date="1. dubna 2026" >}}

- Seznam zemí se správně řadí pro všechny jazyky, včetně znaků s diakritikou

{{< /changelog >}}

{{< changelog version="5.0" date="30. března 2026" >}}
- 14denní bezplatná zkušební doba s jednorázovým nákupem pro trvalé odemknutí
- Obrazovka platby po skončení zkušební doby s možností obnovení předchozích nákupů
- Odpočet zkušební doby na hlavní obrazovce
- Hodinky s Wear OS zobrazí zprávu o odemknutí z telefonu, když zkušební doba vyprší
- Odstraněna duplikace překrývajících se ENTSO-E TimeSeries
- Verze aplikace zobrazena ve spodní části obrazovky nastavení
{{< /changelog >}}
