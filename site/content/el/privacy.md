## Επισκόπηση

Το SweetSpot έχει σχεδιαστεί με γνώμονα την ιδιωτικότητα. Η εφαρμογή δεν συλλέγει ούτε αποθηκεύει προσωπικά δεδομένα. Δεν υπάρχουν λογαριασμοί χρηστών, αναλυτικά στοιχεία ή παρακολούθηση χρήσης. Μια προαιρετική λειτουργία σας επιτρέπει να μοιράζεστε ανώνυμα στατιστικά API — δείτε τις λεπτομέρειες παρακάτω.

## Επεξεργασία δεδομένων

Το SweetSpot ανακτά τιμές ηλεκτρικής ενέργειας επόμενης ημέρας από δημόσια API:

- **ENTSO-E Transparency Platform** — η κύρια πηγή για όλες τις 43 ευρωπαϊκές ζώνες προσφοράς
- **Spot-Hinta.fi** — εναλλακτική πηγή για τις σκανδιναβικές και βαλτικές ζώνες
- **Energy-Charts** — εναλλακτική πηγή για 15 ευρωπαϊκές ζώνες
- **EnergyZero** — εναλλακτική πηγή για την Ολλανδία
- **aWATTar** — εναλλακτική πηγή για την Αυστρία και τη Γερμανία

Αυτά τα αιτήματα API περιέχουν μόνο το αναγνωριστικό ζώνης προσφοράς και το εύρος ημερομηνιών. Δεν περιλαμβάνονται προσωπικές πληροφορίες.

## Τοπική αποθήκευση

Τα δεδομένα τιμών αποθηκεύονται τοπικά στη συσκευή σας για τη μείωση των κλήσεων API και την επιτάχυνση των αποτελεσμάτων. Η διαμόρφωση των συσκευών σας (ονόματα, διάρκειες, εικονίδια) και οι ρυθμίσεις (χώρα, ζώνη, γλώσσα) αποθηκεύονται επίσης τοπικά στη συσκευή σας.

Στο Wear OS, τα δεδομένα συσκευών και οι ρυθμίσεις συγχρονίζονται μεταξύ τηλεφώνου και ρολογιού μέσω του Wearable Data Layer API. Αυτή η επικοινωνία παραμένει στις τοπικές σας συσκευές και δεν περνά μέσω κανενός εξωτερικού διακομιστή.

## Χωρίς αναλυτικά

Το SweetSpot δεν περιλαμβάνει SDK αναλυτικών, αναφορές σφαλμάτων ή παρακολούθηση χρήσης. Η εφαρμογή δεν κάνει αιτήματα δικτύου πέρα από την ανάκτηση τιμών ηλεκτρικής ενέργειας από τα δημόσια API που αναφέρονται παραπάνω (και την προαιρετική αναφορά στατιστικών, αν είναι ενεργοποιημένη).

## Προαιρετικά στατιστικά API

Μπορείτε να συμμετάσχετε στην κοινοποίηση ανώνυμων στατιστικών αξιοπιστίας API. Όταν είναι ενεργοποιημένη, η εφαρμογή αποστέλλει περιοδικά μεμονωμένες εγγραφές αιτημάτων για κάθε πηγή δεδομένων και ζώνη προσφοράς στον διακομιστή μας. Αυτά τα δεδομένα περιέχουν:

- Χρονοσφραγίδα του αιτήματος API
- Αναγνωριστικό ζώνης προσφοράς (π.χ. "NL", "DE-LU")
- Όνομα πηγής δεδομένων (π.χ. "ENTSO-E", "EnergyZero")
- Τύπο συσκευής (τηλέφωνο ή ρολόι)
- Αν το αίτημα πέτυχε ή απέτυχε
- Κατηγορία σφάλματος σε περίπτωση αποτυχίας (π.χ. "χρονικό όριο", "σφάλμα διακομιστή")
- Αριθμό έκδοσης εφαρμογής

Αυτά τα δεδομένα **δεν** περιέχουν αναγνωριστικά συσκευής, τοποθεσία, δεδομένα τιμών ή άλλες προσωπικές πληροφορίες. Χρησιμοποιούνται αποκλειστικά για τη βελτίωση της αξιοπιστίας των πηγών δεδομένων και της προεπιλεγμένης σειράς.

Αυτή η λειτουργία είναι απενεργοποιημένη από προεπιλογή. Μπορείτε να την ενεργοποιήσετε ή να την απενεργοποιήσετε ανά πάσα στιγμή στις Ρυθμίσεις > Για προχωρημένους.
