{{< changelog version="5.2" date="2 de abril de 2026" >}}

- El selector de iconos ahora muestra el nombre del icono seleccionado, traducido a los 25 idiomas admitidos

{{< changelog version="5.1.5" date="1 de abril de 2026" >}}
- Símbolos de depuración nativos incluidos en los paquetes de lanzamiento para mejorar los informes de fallos en Play Store
{{< /changelog >}}

{{< changelog version="5.1.4" date="1 de abril de 2026" >}}
- Diálogo de agradecimiento tras suscribirse a SweetSpot
- Corregido un destello del idioma anterior al cambiar el idioma de la app en ajustes
{{< /changelog >}}

{{< changelog version="5.1.3" date="1 de abril de 2026" >}}
- Ajustes reorganizados: fuentes de datos, caché y opciones de desarrollador están ahora en una sección Avanzado
- La lista de países se ordena correctamente en todos los idiomas, incluidos los caracteres acentuados
- Mejora de la naturalidad y gramática en varios idiomas
{{< /changelog >}}

{{< changelog version="5.1.2" date="30 de marzo de 2026" >}}
- Añadida opción de suscripción anticipada en Ajustes durante el periodo de prueba
{{< /changelog >}}

{{< changelog version="5.1.1" date="30 de marzo de 2026" >}}
- Corregidas las versiones de teléfono y reloj para que tengan códigos de versión distintos para la subida a Play Console
{{< /changelog >}}

{{< changelog version="5.0" date="30 de marzo de 2026" >}}
- Periodo de prueba gratuito de 14 días con suscripción anual para seguir usando la app
- Pantalla de pago tras finalizar el periodo de prueba con opción de restaurar suscripción
- Cuenta atrás del periodo de prueba en la pantalla principal
- El reloj Wear OS muestra un mensaje para suscribirse desde el teléfono cuando expira el periodo de prueba
- Corregida la deduplicación de TimeSeries superpuestos de ENTSO-E
- Versión de la aplicación visible en la parte inferior de la pantalla de ajustes
{{< /changelog >}}

{{< changelog version="4.1" date="30 de marzo de 2026" >}}
- Estadísticas anónimas opcionales de fiabilidad de API para mejorar la calidad de las fuentes de datos
- Mejora del manejo de errores en las cinco fuentes de datos
{{< /changelog >}}

{{< changelog version="4.0" date="28 de marzo de 2026" >}}
- Nuevo identificador de aplicación: `today.sweetspot`
- Mejoras en el sitio web y validación del sitio
{{< /changelog >}}

{{< changelog version="3.5" date="28 de marzo de 2026" >}}
- Añadido aWATTar como fuente de datos de respaldo para Austria y Alemania
- Formato de precios en EUR adaptado a la configuración regional
- Mejora de la calidad de las traducciones en 25 idiomas
- La pantalla de resultados ahora se actualiza completamente cada 60 segundos
- Dependencias actualizadas a las últimas versiones estables

{{< changelog version="3.4" date="26 de marzo de 2026" >}}

- Borrar caché de precios desde ajustes y botón de actualización en la pantalla de resultados
- La zona de precios ahora se muestra en la pantalla de resultados
- Mejora de la gramática en textos traducidos que contienen números
{{< /changelog >}}

{{< changelog version="3.3" date="26 de marzo de 2026" >}}
- Los ajustes ahora muestran el nombre del idioma del sistema

{{< changelog version="3.2" date="5 de marzo de 2026" >}}

- Preferencias de fuentes de datos configurables con arrastrar para reordenar
- Diseño reorganizado de la pantalla de ajustes
