---
title: "Histórico de versões"
description: "Histórico de versões e notas de lançamento do SweetSpot."
---

{{< changelog version="5.1.3" date="1 de abril de 2026" >}}
- Definições reorganizadas: fontes de dados, cache e opções de programador estão agora numa secção Avançado
- A lista de países ordena corretamente para todos os idiomas, incluindo caracteres acentuados
- Melhoria da naturalidade e gramática em vários idiomas
{{< /changelog >}}

{{< changelog version="5.1.2" date="30 de março de 2026" >}}
- Adicionada opção de desbloqueio antecipado nas Definições durante o período experimental
{{< /changelog >}}

{{< changelog version="5.1.1" date="30 de março de 2026" >}}
- Corrigidas as compilações do telemóvel e relógio para terem códigos de versão distintos para carregamento no Play Console
{{< /changelog >}}

{{< changelog version="5.0" date="30 de março de 2026" >}}
- Período experimental gratuito de 14 dias com compra única para desbloqueio permanente
- Ecrã de pagamento após o fim do período experimental com opção de restaurar compras anteriores
- Contagem decrescente do período experimental apresentada no ecrã principal
- O relógio Wear OS mostra uma mensagem para desbloquear a partir do telemóvel quando o período experimental expira
- Corrigida a deduplicação de TimeSeries ENTSO-E sobrepostas
- Versão da aplicação apresentada na parte inferior do ecrã de definições
{{< /changelog >}}

{{< changelog version="4.1" date="30 de março de 2026" >}}
- Estatísticas anónimas opcionais de fiabilidade da API para ajudar a melhorar a qualidade das fontes de dados
- Melhoria do tratamento de erros em todas as cinco fontes de dados
{{< /changelog >}}

{{< changelog version="4.0" date="28 de março de 2026" >}}
- Novo identificador da aplicação: `today.sweetspot`
- Melhorias e validação do website
{{< /changelog >}}

{{< changelog version="3.5" date="28 de março de 2026" >}}
- Adicionado aWATTar como fonte de dados de recurso para a Áustria e a Alemanha
- Formatação de moeda EUR adaptada às configurações regionais
- Melhoria da qualidade das traduções em 25 idiomas
- Corrigido o ecrã de resultados para atualizar completamente a cada 60 segundos
- Dependências atualizadas para as versões estáveis mais recentes
{{< /changelog >}}

{{< changelog version="3.4" date="26 de março de 2026" >}}
- Adicionado Energy-Charts como fonte de dados de recurso para 15 zonas europeias
- Limpar cache de preços nas definições e botão de atualização no ecrã de resultados
- A zona de preço é agora apresentada no ecrã de resultados
- Melhoria da gramática para textos traduzidos com números
{{< /changelog >}}

{{< changelog version="3.3" date="26 de março de 2026" >}}
- As definições apresentam agora o nome do idioma do sistema
- 25 idiomas agora suportados
{{< /changelog >}}

{{< changelog version="3.2" date="5 de março de 2026" >}}
- Adicionados 21 idiomas europeus incluindo neerlandês, alemão e francês (25 no total)
- Seletor de idioma em ecrã inteiro
- Preferências de fontes de dados configuráveis com arrastar para reordenar
- Disposição do ecrã de definições reorganizada
{{< /changelog >}}

{{< changelog version="3.1" date="4 de março de 2026" >}}
- ENTSO-E é agora a fonte principal para os Países Baixos (EnergyZero como recurso)
- Suporte para resolução de preços inferior a uma hora (15 minutos)
{{< /changelog >}}

{{< changelog version="3.0" date="3 de março de 2026" >}}
- Integração com a API da ENTSO-E Transparency Platform para todas as zonas europeias
- Suporte multi-zona: 30 países, 43 zonas de licitação
- Seleção de país e zona com deteção automática
- Preços guardados localmente para carregamento mais rápido
{{< /changelog >}}

{{< changelog version="2.3" date="3 de março de 2026" >}}
- Licenciado sob GPL v3
- Gesto preditivo de retrocesso no Android 13+
- Melhorias de acessibilidade (suporte para leitor de ecrã)
- Cache desatualizada como recurso quando a atualização falha
- Melhorias de estabilidade e correções de erros
{{< /changelog >}}

{{< changelog version="2.2" date="2 de março de 2026" >}}
- Tamanho da aplicação reduzido
- Melhorias de segurança e estabilidade
- Correções de erros
{{< /changelog >}}

{{< changelog version="2.1" date="2 de março de 2026" >}}
- APK Wear OS incluído nos lançamentos
- Apresentação de tempo relativo melhorada (arredondado ao minuto mais próximo)
{{< /changelog >}}

{{< changelog version="2.0" date="2 de março de 2026" >}}
- Aplicação complementar Wear OS com sincronização automática
- Consulte preços a partir do pulso com eletrodomésticos guardados
{{< /changelog >}}

{{< changelog version="1.2" date="2 de março de 2026" >}}
- Corrigido um problema de temporização quando o período mais barato começa imediatamente
- Adicionado aviso sobre preços spot
{{< /changelog >}}

{{< changelog version="1.1" date="2 de março de 2026" >}}
- Texto da interface e disposição do ecrã de definições melhorados
- Ícone da aplicação refinado
{{< /changelog >}}

{{< changelog version="1.0" date="2 de março de 2026" >}}
- Lançamento inicial
- Seletor de duração com roda de deslocamento e botões de duração rápida (1–6 horas)
- Eletrodomésticos configuráveis com nomes, ícones e durações personalizados
- Ecrã de resultados com discriminação de custos por intervalo de tempo
- Gráfico de barras com os próximos preços e o período mais barato destacado
{{< /changelog >}}
